# Civic Connect - Admin Panel

## Setup
1. `npm install`
2. `npm run dev`

## Features (UI Mock)
- **Dashboard**: View statistics and charts.
- **Issue List**: Filter and search issues.
- **Map View**: Geospatial visualization of issues.
- **Login**: Mock authentication (admin@example.com / admin123).

## Tech Stack
- React + Vite
- Tailwind CSS
- Recharts
- Leaflet / React-Leaflet
